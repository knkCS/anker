# MenuCell

Renders row actions as a dropdown menu or inline icon buttons based on a configurable threshold. When the number of actions exceeds `menuThreshold` (default 1), all actions collapse into a three-dot dropdown. Stops event propagation to avoid triggering `onRowClick`.
