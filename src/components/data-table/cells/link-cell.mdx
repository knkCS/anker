# LinkCell

Renders a router-aware link for in-app navigation using react-router-dom. Composes Chakra's `Link` (styling) with react-router-dom's `Link` (routing) via `asChild`. For external URLs, use `UrlCell` instead. Stops event propagation to avoid triggering `onRowClick`.
